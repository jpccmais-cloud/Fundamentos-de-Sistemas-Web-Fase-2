# Petshop Aurora - Projeto Acadêmico (Fase 2)

Este projeto representa a Fase 2 do sistema web para o Petshop Aurora. O foco desta fase foi aprimorar a interface e a funcionalidade do projeto da Fase 1, incorporando:

- **Estilização e Responsividade:** Uso do framework **Bootstrap 5** e CSS personalizado.
- **Interatividade:** Implementação de **JavaScript** para funções temporais e validação de formulário.
- **Funcionalidades:** Adição de um **Carrossel** na página inicial e um **Formulário Completo de Agendamento** (Cliente e Pet).
- **Acessibilidade:** Garantia do uso do atributo  em todas as imagens.
